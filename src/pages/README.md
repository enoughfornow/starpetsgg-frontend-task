# /src/pages - Слой страниц приложения

**Страница не содержит какаой либо бизнес-логики, это контейнер для отображения!!!**

Страница - это точка входа, площадка на которой расположены разные фичи, виджеты и ентити, так же в файлах index расположен роут по которому эта страница доступна, если роут вложенный, то в папку страницы добавляется папка pages с дочерними роутами


У компонентов страницы префикс `P`, например `PConvert.vue`
