# src/features - слой с фичами 

Фича - это какая то полезная пользователю функция приложения, зачастую такая функция может быть "размазана" по разным страницам, виджетам

Фича содержит в себе взаимодействие различных entities(src/entities), у фичи может быть свое хранилище для сохранения состояния и взаимодействия с фичей, в фиче могут быть расположены различные компоненты которые касаются этой фичи
